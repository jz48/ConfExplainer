# ConfExplainer

Data and code for paper: "Is Your Explanation Reliable: Confidence-Aware Explanation on Graph Neural Networks".

## Requirements
- Python                   3.8
- torch                    1.12.1
- torch-geometric          2.1.0.post1
- torch-geometric-temporal 0.54.0
- torch-scatter            2.0.9
- torch-sparse             0.6.15

## Dataset
